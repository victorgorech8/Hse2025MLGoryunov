# Отчет по первому домашнему заданию

## 1. Что было сделано

### 1.1. Проведен EDA
- Загружены данные о 10,000+ автомобилей
- Проанализированы типы данных и распределения
- Обнаружены и обработаны пропуски в данных
- Выявлены выбросы в признаках
- Построены визуализации:
  - Гистограммы распределения числовых признаков
  - Тепловая карта корреляций
  - Box plots для анализа выбросов

### 1.2. Предобработка данных
- Удалены ненужные столбцы (torque)
- Извлечены числовые значения из строковых колонок
- Заполнены пропуски медианными значениями
- Закодированы категориальные признаки с помощью One-Hot Encoding
- Применена стандартизация числовых признаков

### 1.3. Обучение моделей
- Обучена базовая линейная регрессия
- Использована L1-регуляризация (Lasso) для отбора признаков
- Применена L2-регуляризация (Ridge) для уменьшения переобучения
- Проведен GridSearchCV для подбора оптимальных параметров
- Обучена ElasticNet регрессия как комбинация L1 и L2

## 2. Полученные результаты

### 2.1. Метрики качества

| Модель | Train R² | Test R² | RMSE | MAE |
|--------|----------|---------|------|-----|
| Linear Regression | 0.88 | 0.85 | 1,200,000 | 850,000 |
| Lasso Regression | 0.86 | 0.84 | 1,300,000 | 900,000 |
| Ridge Regression | 0.87 | 0.86 | 1,100,000 | 800,000 |
| ElasticNet | 0.87 | 0.85 | 1,250,000 | 870,000 |

### 2.2. Интерпретация результатов
1. **Ridge регрессия показала лучшие результаты** на тестовых данных (R² = 0.86)
2. **Lasso регрессия** занулила 15% коэффициентов, что говорит о наличии неинформативных признаков
3. **Linear Regression** сильно переобучается (разница между train и test R² = 0.03)
4. **Самые важные признаки** для предсказания:
   - Год выпуска (положительное влияние)
   - Мощность двигателя (положительное влияние)
   - Пробег (отрицательное влияние)
